# Ассемблер и Интерпретатор для УВМ (Учебной Вычислительной Машины)

## Описание

Этот репозиторий содержит реализацию ассемблера и интерпретатора для условной Учебной Вычислительной Машины (УВМ), вариант 24. Проект включает в себя:

*   Ассемблер, преобразующий исходный код на человекочитаемом языке ассемблера (с использованием мнемоник) в бинарный формат.
*   Интерпретатор, выполняющий бинарный код на виртуальной машине.
*   Скрипты для запуска тестов и сохранения результатов.

## Язык ассемблера

Язык ассемблера спроектирован в соответствии с традиционными ассемблерами. Каждая строка содержит мнемонику команды, за которой следуют аргументы. Комментарии начинаются с символа `#`. Язык поддерживает все команды, определённые в спецификации УВМ.

### Поддерживаемые команды

*   `LOAD_CONST <регистр_B>, <константа_C>`
    *   Загружает константу `C` в регистр `B`.
    *   Формат: A (8 бит), B (3 бита), C (13 бит).
    *   Код операции: 234 (0xEA).
*   `READ_MEM <смещение_B>, <регистр_результата_C>, <регистр_базы_D>`
    *   Читает значение из памяти по адресу `регистр_базы + смещение` и сохраняет в `регистр_результата`.
    *   Формат: A (8 бит), B (13 бит), C (3 бита), D (3 бита).
    *   Код операции: 102 (0x66).
*   `WRITE_MEM <регистр_данных_B>, <регистр_адреса_C>`
    *   Записывает значение из `регистр_данных` в память по адресу, указанному в `регистр_адреса`.
    *   Формат: A (8 бит), B (3 бита), C (3 бита).
    *   Код операции: 90 (0x5A).
*   `SHIFT_RIGHT <регистр_значения_B>, <регистр_сдвига_C>, <смещение_D>, <регистр_базы_E>`
    *   Выполняет логический сдвиг вправо значения из `регистр_значения` на `регистр_сдвига + смещение` битов и записывает результат в память по адресу `регистр_базы`.
    *   Формат: A (8 бит), B (3 бита), C (23 бита), D (13 бит), E (3 бита).
    *   Код операции: 224 (0xE0).
